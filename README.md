# Guess the Number Game 🎮

A simple Python guessing game.

## Features
- Random number between 1 and 100
- Unlimited guesses
- Type `quit` to exit
- Gives hints if the guess is too high or too low

## How to Run

```bash
python guess_the_number.py
```

## Example

Enter Your target or quit: 50
----Your Target Is Small Try A Big Number----

Enter Your target or quit: 75
----You Win----
```
